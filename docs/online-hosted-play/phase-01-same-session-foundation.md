# Phase 1: Same-session foundation

Goal: ship the first playable online hosted-play slice for a real live
character.

Deliverables:
- hosted runtime module
- named `GameServer` hosted-play seam
- owner status and enable or disable API
- game-menu UI control surface
- first automation bridge that reuses the ambient progression brain
- manual-pause safety

Out of scope:
- persistence
- login auto-resume
- party support
- social shell
- LLM integration
